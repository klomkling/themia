# Themia 0.2.0 — Tier 0 (Tooling) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended)
> or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`)
> syntax for tracking.

**Goal:** Port and unify the build-time tooling family into Themia — `Themia.Generators.Abstractions`,
`Themia.SourceGenerator` (Idevs DI-gen + zenity-v1 mediator-gen on one model), and `Themia.Analyzers`
(the two type-independent rules) — all `netstandard2.0`, all tests green, clean build.

**Architecture:** This is the first of four tiers in the 0.2.0 framework rename (spec:
`docs/superpowers/specs/2026-06-02-themia-0.2.0-framework-rename-design.md`, esp. §3b). It is a
**port**, not net-new code: existing, tested Roslyn source from three trees is relocated, namespaces
and diagnostic IDs renamed (`Idevs.*`/`Zenity.*` → `Themia.*`, `IDEVSGEN → THEMIA`), the two DI
generators reconciled (Idevs canonical, drop v1's), and v1's mediator generator grafted onto the
shared abstractions. `Themia.Analyzers.CodeFixes` and analyzer rules `THEMIA100`/`THEMIA103` are
**deferred to Phase 3** (their target types — UoW, `ISequenceProvider` — don't exist in 0.2.0).

**Tech Stack:** .NET 10 SDK; `netstandard2.0` Roslyn components; `Microsoft.CodeAnalysis.CSharp`;
incremental generators (`IIncrementalGenerator`) + `DiagnosticAnalyzer`; xUnit + `Verify.SourceGenerators`
snapshot tests + Roslyn SDK testing packages; central package management; `PublicAPI.*` tracking.

`[Scoped] class FooService : IFooService` + a mediator handler, runs **both** generators, and asserts

```csharp
// tests/Themia.SourceGenerator.Tests/EndToEnd/ConsumerCompilationTests.cs
using Microsoft.CodeAnalysis;
using Microsoft.CodeAnalysis.CSharp;
using Xunit;

namespace Themia.SourceGenerator.Tests.EndToEnd;

public class ConsumerCompilationTests
{
    [Fact]
    public void Both_generators_run_in_one_driver_and_emit_registration()
    {
        const string source = """
            using Themia.DependencyInjection;
            [assembly: GenerateServiceRegistrations]
            namespace Consumer;
            public interface IFooService { }
            [Scoped] public class FooService : IFooService { }
            """;
        var compilation = CSharpCompilation.Create("Consumer",
            new[] { CSharpSyntaxTree.ParseText(source) },
            Basic.Reference.Assemblies.Net80.References.All, // or the harness's metadata refs
            new CSharpCompilationOptions(OutputKind.DynamicallyLinkedLibrary));

        var driver = CSharpGeneratorDriver.Create(
            new ServiceRegistrationGenerator().AsSourceGenerator(),
            new MediatorHandlerGenerator().AsSourceGenerator());

        driver.RunGeneratorsAndUpdateCompilation(compilation, out var output, out var diagnostics);

        var generated = string.Join("\n", output.SyntaxTrees.Select(t => t.ToString()));
        Assert.DoesNotContain(diagnostics, d => d.Severity == DiagnosticSeverity.Error);
        Assert.Contains("AddScoped<global::Consumer.IFooService, global::Consumer.FooService>", generated);
    }
}
```
