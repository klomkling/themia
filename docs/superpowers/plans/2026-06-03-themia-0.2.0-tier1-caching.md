### Transform rules (apply to every ported file)

1. **Namespaces:** `Zenity.Core.Caching` → `Themia.Caching` (all sub-namespaces: `.Abstractions`,
   `.Configuration`, `.Extensions`, `.Internal`, `.Providers`, `.Serialization`).
2. **Cross-reference usings:** `using Zenity.Core.Caching.*` → `using Themia.Caching.*` (the only one is
   `using Zenity.Core.Caching.Internal;`).
3. **Identifier rebrand — every `Zenity` identifier → `Themia`:** types `IZenityCacheProvider → IThemiaCacheProvider`,
   `IZenityCachingBuilder → IThemiaCachingBuilder`, `ZenityCachingBuilder → ThemiaCachingBuilder`; the public
   entry method `AddZenityCaching → AddThemiaCaching` (all overloads). Update every reference.
4. **Drop ImplicitUsings-covered usings** (`System`, `System.Collections.Generic`, `System.Linq`,
   `System.Threading`, `System.Threading.Tasks`); keep all others (e.g. `System.Text.Json`,
   `Microsoft.Extensions.*`, `StackExchange.Redis`, `MessagePack`).
5. **No logic changes.** Verbatim port + transforms only.
